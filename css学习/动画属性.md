### 2. 单个变换属性
```
element {
  scale: 2; /* 纵横比 */
  rotate: 30deg; /* 角度 */
  translate: -50% -50%; /* 平移 */
}

.target {
  transform: translateX(50%) rotate(30deg) scale(1.2);
}
```

固定顺序：平移——>旋转——>缩放
